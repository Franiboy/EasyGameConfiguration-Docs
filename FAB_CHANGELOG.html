v1.3 - Stability & Memory

Support starts from Unreal Engine 5.5 onwards.

Bug Fixes

  • Fixed Vector serialization: Y and Z components of default values incorrectly used the X value.
  • Fixed null-pointer crash when setting a property with uninitialized category or key.
  • Fixed Vector2D editor refresh incorrectly triggering Vector BP node recompilation.
  • Fixed FLinearColor default alpha from 0 to 1 (colors were invisible by default).
  • Fixed false "Failed to create save game object" error during anti-cheat detection.
  • Fixed stale config data persisting between sessions (CleanUp now empties all caches).
  • Fixed unnecessary FString dereference in legacy setters and GetSettingIntern.
  • Added null-check before GenerateHashOfSaveGame to prevent crash on missing data.

Memory Management

  • Replaced raw pointer allocations with TUniquePtr across 91 Blueprint function call-sites -- eliminates memory leaks.
  • Converted 18 heap-allocated TMap delegate registrations to stack variables.
  • Changed GetCurrentPropertysFromFile/SaveGame return types from pointer to value (no heap allocation).
  • Added NativeDestruct() to all 9 widget classes to prevent widget memory leaks.
  • Initialized all SaveGameObject and TempWidget pointers to nullptr.

Quality of Life

  • Refactored CallOnChangeDelegate to use safe intermediate lookups.
  • Added null-checks to all Get{Type}Widget functions.

v1.2 - Multi-Engine Support

  • Added support for Unreal Engine 5.1 and 5.2.
  • Updated packaging process for multi-version distribution.

v1.1 - Anti-Cheat, Reset & Polish

New Features

  • Anti-Cheat system: Hash-based validation for save game data with configurable per-property cheat detection values.
  • Reset functionality: Reset individual properties or all properties at once to their default values.
  • Blueprint keyword "EGC" registered for all methods -- find any node by searching "EGC" in the Blueprint editor.

Improvements

  • Simplified property struct internals for cleaner data handling.
  • Improved save game process for plugin settings.
  • Code cleanup: removed unnecessary code and unused .ini files.
  • Added copyright headers to all source files.

v1.0 - Initial Release

Core Features

  • 9 supported data types: Boolean, Integer, Float, String, Vector2D, Vector, Color, Rotator, Transform.
  • Context-free Blueprint access: Get/Set any property from anywhere with a single node.
  • Dual API: Enum-based (recommended) and string-based (legacy) access patterns.
  • Auto-generated .ini configuration files.
  • Save game support for properties not editable by users.
  • Delegate system: Subscribe to property changes per type.
  • Automatic UI widget rendering for all defined properties with per-property style overrides.
  • Editor integration: Custom Project Settings tab, Blueprint editor buttons, configurable shortcuts.
  • Property validation with allowed/forbidden value constraints.
  • Multi-save support for per-player or per-server configurations.
  • Platforms: Win64, Android, Linux, LinuxArm64.
